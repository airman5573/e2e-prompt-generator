Guide

This page is an example for testing the E2E Prompt Builder Extension.

How to use the Extension:
1. Activate the extension
2. Hover over elements to see highlights
3. Use arrow keys (‚Üë‚Üì) to navigate parent/child elements
4. Press Space to open the prompt modal and insert selectors
5. Type text and press Enter to proceed to the next step
6. Press ESC to add multiple elements to the same step
7. Use the Copy button to copy the entire prompt

All elements have IDs, optimized for Extension testing!
